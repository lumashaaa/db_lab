Эксперимент A — потеря реплики node3

1. Остановить node3:
   docker stop node3

2. Вставить данные в node1:
   docker exec node1 clickhouse-client --query "
   INSERT INTO repl.events SELECT now(), 'test_a', rand()%1000, 'exp_a'
   FROM numbers(1000);"

3. Проверить node2:
   docker exec node2 clickhouse-client --query "SELECT count() FROM repl.events;"

4. Поднять node3:
   docker start node3

5. Дождаться синхронизации и проверить queue_size:
   docker exec node3 clickhouse-client --query "
   SELECT replica_name, queue_size, last_queue_update
   FROM system.replicas WHERE table = 'events';"
